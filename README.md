> Step inside **The Farmstand 3D** — an immersive, WebXR-powered virtual marketplace where cannabis vendors showcase products in a 3D environment you can explore from any browser.

## About the Project

The Farmstand 3D reimagines online cannabis shopping as a spatial, explorable experience. Built on top of [A-Frame](https://aframe.io/), a web framework for building 3D/AR/VR experiences, the marketplace runs directly in the browser — no app download required.

**Live site:** [marijuanaunion.com/marketplace/](https://marijuanaunion.com/marketplace/)

### Features

- **3D Product Displays** — Browse products placed in a virtual farmstand environment
- **WebXR Ready** — Works in VR headsets (Quest, etc.) as well as flat-screen browsers
- **Vendor Booths** — Each vendor gets a dedicated 3D space to showcase their offerings
- **Mobile Friendly** — Touch-based navigation on phones and tablets
- **No Install Required** — Pure web technology, runs in Chrome, Firefox, Edge, Safari

## Getting Started

```bash
# Clone this repo
git clone https://github.com/nicedreamzapp/the-farmstand-3d.git
cd the-farmstand-3d

# Open index.html in any modern browser — no build step needed
# For VR, serve over HTTPS (required by WebXR)
python -m http.server 8443
```
